# Tactical Ottoine Monster Hunter World XXIII

A console RPG. Work in progress.

Needs to connect to a running server. Setup server connection config in `db.json`.

### Install dependencies
```sh
$ npm install
```

### Run game
```sh
$ ./game
```
or
```sh
$ node game
```

### Run server
```sh
$ ./server
```
or
```sh
$ node server
```
